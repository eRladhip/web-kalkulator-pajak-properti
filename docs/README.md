## Dokumentasi Singkat

Panduan singkat aplikasi Kalkulator Pajak Properti.

- Struktur proyek: `src/components`, `src/utils`, `src/pages`, `public`, `docs`.
- Cara jalankan: `npm install` lalu `npm run dev`.
- Aturan perhitungan mengikuti ringkasan UU HPP & PMK 2024-2025 seperti dicantumkan di README.

Untuk integrasi lebih lanjut (deploy Netlify/ Vercel), paket sudah menggunakan Vite sehingga tinggal build dan deploy folder `dist`.
